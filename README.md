# iketeru_lp
これは、イケテルシステムのWordpressをGatsbyに変更するプロジェクトです

# 必要条件  
Node.js 16.15.0

# システム構成
- Gatsby.js
  - React

# 起動方法
`npm install`  
その後 `npm start` で開発サーバーが起動します。  
production環境で確認したい場合、  
`npm run build` でビルドした後に  
`npm run serve` を叩いてください。

# ディレクトリ構成
- src/components/common
  - どこでも使える共通コンポーネントを格納
- src/components/p-lp
