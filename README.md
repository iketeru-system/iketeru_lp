# イケテルシステムのホームページ

## システム構成
- Gatsby.js
  - React

## 環境構築手順
### システム要件
- Node.js 16.15.0

### 手順
1. `npm i` を実行します（初回のみ）。
2. `npm start` を実行します。
3. `localhost:8000` にアクセスすると、開発サーバーのページが表示されます。
